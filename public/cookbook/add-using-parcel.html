Issue

We need to add Scrawl-canvas functionality to a HTML5/CSS/JS project that uses the Parcel web application bundler.

Solution

Include Scrawl-canvas in the project using Yarn or NPM, then import it into your Javascript code as normal:

Console instructions
$> yarn add scrawl-canvas
$> yarn install

index.js
import * as scrawl from 'scrawl-canvas';

const canvas = scrawl.findCanvas('mycanvas');

// Namespacing boilerplate
const namespace = 'my-demo';
const name = (n) => `${namespace}-${n}`;

scrawl.makeBlock({
  name: name('block'),
  start: ['center', 'center'],
  handle: ['center', 'center'],
  dimensions: ['50%', '50%'],
});

scrawl.render();
